# MotionAlongCurve
A Unity project demonstrating motion along a 3D Bézier path using arc-length parameterization. Includes adaptive distance-table generation, speed control tied to animation pacing, editable control points, and smooth orientation using Frenet frames for natural character movement.
To test it, run the CurveGeneration Scene, this program mixes the unity editor with the the simulation execution, to change the trajectory drag the red control points or the white weight points, the agent only moves in the 2D projection of the curve.

Unity Version: 2022.3.4f1
